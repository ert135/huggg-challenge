# huggg-challenge

## Description
Includes 4 endpoints

- /brand
- /brand/:id

- /product
- /product/:id

- /store
- /store/:id
- /store/byProductId/:id

## Installation

```bash
$ npm install
$ npm run start
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests -- not completed
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
